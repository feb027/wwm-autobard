SAMPLE SONGS
============

This folder contains sample songs you can use to test WWM Auto-Bard.

Included:
- Twinkle_Twinkle.json - Simple melody to test the app

Finding More Songs:
- Sky Sheets: https://sky-music.github.io/
- MIDI Files: https://bitmidi.com/ or https://www.midiworld.com/

Supported Formats:
- .mid / .midi - Standard MIDI files
- .json - Sky: Children of the Light sheet format
- .txt / .skysheet - Sky sheet text format
